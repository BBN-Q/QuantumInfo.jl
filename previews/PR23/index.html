QuantumInfo.ket — Function

ket

Returns a vector corresponding to a particular basis unit vector in some Hilbert space.

The vector can be specified in a number of different ways:

  • by a string and a base
  • by a integer and a dimension

If the vector is specified by a string and a base, the dimension is inferred from the length of the string. The base is assumed to be 2 unless explicitly specified.

If the basis element is specified by an integer, the dimension is assumed to be 2 unless explicitly specified. The integer values allowed are 0 through the dimension minus 1.

Optionally, the type of the elements of the vector can be specified in the first argument.

QuantumInfo.bra — Function

bra

Returns a dual vector (row vector) corresponding to a particular basis unit vector in some Hilbert space.

See ket for details on how the corresponding vector may be specified.

source
QuantumInfo.ketbra — Function

ketbra

Returns the outer product between a dual vector (row vector) and a vector, each corresponding to a particular basis unit vector in some Hilbert space. This can also be interpreted as a matrix unite element specified by two unit vectors for a row and a column.

See ket for details on how the corresponding vectors may be specified.

QuantumInfo.projector — Function

projector(v)

Computes the rank-1 projector operator corresponding to a given vector. If more than one vector is given, the projector into the space spanned by the vectors is computed.

QuantumInfo.depol — Function

depol(d, p=1.0)

depol return a superoperator that replaces the input with a maximally mixed state with probability p, and leaves it unchanged with probability (1-p).

source
QuantumInfo.unitalproj — Function

unitalproj(m)

Given a superoperator j in a Liouville representation, unitalproj extracts the closest superoperator (in Frobenius norm) that is unital. The result may not be completely positive.
